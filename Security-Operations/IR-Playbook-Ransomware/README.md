# 🛡️ AI-Assisted Incident Response Playbook: Ransomware Attack
**Category:** Incident Response / GRC | **Tools:** ChatGPT, NIST SP 800-61 Rev. 2 Framework | **Target:** CloudNova Solutions (IaaS Provider)

## 📖 Project Overview
Developed a comprehensive Incident Response (IR) Playbook for **CloudNova Solutions** following a massive ransomware outbreak attributed to the "DarkCipher" group. This project demonstrates how to utilize Generative AI to rapidly draft, refine, and customize professional security documentation aligned with industry-standard frameworks.

## 🎯 Technical Objectives
* **Scenario Analysis:** Extracted critical indicators of compromise (IoCs) and attack vectors from a multi-site data center breach.
* **AI Prompt Engineering:** Utilized targeted prompts to generate an IR template and refined it using specific incident data (Phishing entry point, unpatched monitoring tools).
* **Framework Alignment:** Aligned the playbook with the **NIST Incident Response Life Cycle** (Preparation, Detection/Analysis, Containment/Eradication, and Post-Incident Activity).
* **Stakeholder Management:** Defined the specific roles for the CISO, Legal, and Customer Relations teams to manage a 70+ client outage.



## 🛠️ The IR Playbook Phases
1. **Preparation:** Hardening third-party tools and phishing awareness training.
2. **Detection & Analysis:** Identifying the "DarkCipher" payload and file encryption patterns.
3. **Containment:** Isolating affected Virtual Machines (VMs) across both data centers.
4. **Eradication:** Removing the malicious macro-enabled Excel documents and patching monitoring tools.
5. **Recovery:** Restoring services from immutable backups and verifying data integrity.
