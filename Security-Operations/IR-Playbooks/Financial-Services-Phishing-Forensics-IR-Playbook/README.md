# 🎣 Financial Services Phishing Forensics & Incident Response Playbook

Welcome to this Incident Response lab focused on investigating and responding to phishing attacks in a financial services environment.

---

## 🔍 Overview
This project covers forensic analysis and incident response procedures for a simulated phishing attack targeting financial systems, including detection, containment, and recovery steps.

---

## 🎯 Objectives
- Investigate phishing attack indicators  
- Perform forensic analysis of compromised systems  
- Contain and eradicate malicious activity  
- Recover affected systems securely  
- Develop structured IR playbook procedures
