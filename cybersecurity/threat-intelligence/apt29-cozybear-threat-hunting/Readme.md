# 🛡️ APT29 (Cozy Bear) Proactive Threat Hunting & TTP Mapping

Welcome to this Threat Hunting Lab focused on analyzing **APT29 (Cozy Bear)** using proactive hunting methodologies and the **MITRE ATT&CK Framework**.

---

## 🔍 Overview
This project investigates APT29 tactics, techniques, and procedures (TTPs), maps them to MITRE ATT&CK, and explores proactive threat hunting strategies for detecting advanced persistent threats.

- Conduct proactive threat hunting analysis  
- Map APT29 behavior to MITRE ATT&CK

- Strengthen threat hunting methodology
