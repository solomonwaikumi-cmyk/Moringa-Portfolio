# 🛡️ APT10 Adversary Analysis & MITRE ATT&CK Mapping

Welcome to this Threat Intelligence Lab focused on analyzing **APT10**, a sophisticated cyber espionage group, through the **MITRE ATT&CK Framework**.

This project explores the tactics, techniques, and procedures (TTPs) used by APT10 and maps their behavior to the MITRE ATT&CK matrix to better understand adversary operations and defensive opportunities.
