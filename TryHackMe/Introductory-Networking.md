**Platform:** [TryHackMe](https://tryhackme.com/p/solomon.mwaura) | **Category:** Network Security

## 📝 Lab Overview
Networking is the lifeblood of security operations. This lab covers the fundamental models and tools required to monitor and secure data in transit.

## 🛠️ Technical Concepts
* **OSI Model:** Deep-dive into the 7 layers, focusing on Layer 3 (Network/Routing) and Layer 4 (Transport/TCP-UDP).
* **TCP/IP Handshake:** Analyzed the 3-way handshake process (SYN, SYN-ACK, ACK) that establishes reliable connections.
* **Encapsulation:** Understanding how data is "wrapped" in headers as it travels down the OSI stack.

## 🔧 Networking Toolset
During this lab, I utilized the following diagnostic tools:
* `Ping`: Testing reachability and latency.
* `Traceroute`: Mapping the path packets take to a destination.
* `Dig / WHOIS`: Investigating domain records and ownership.
