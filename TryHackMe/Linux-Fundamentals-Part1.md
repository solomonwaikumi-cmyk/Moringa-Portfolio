# 🐧 Room: Linux Fundamentals Part 1
**Platform:** [TryHackMe](https://tryhackme.com/p/solomon.mwaura) | **Category:** Operating Systems

## 📝 Description
Linux is the primary OS for security tools and enterprise servers. This room establishes the "command-line first" mindset required for effective security auditing.

## ⌨️ Command Line Proficiency
Mastered essential navigation and file manipulation commands:
* `ls`, `cd`, `pwd`: Navigating the filesystem.
* `cat`, `grep`: Reading and searching through logs for specific strings.
* `find`: Locating sensitive files across the system.

## 🛡️ Security Implications
* **Shell Operators:** Used `>` and `>>` for output redirection, essential for creating audit logs.
* **Process Management:** Learned to identify and manage running services using the terminal.

## 💡 Professional Alignment
This room provides the foundation for **Moringa Lab 01 (System Audit)** and **Lab 06 (Linux Hardening)**, where I performed user account audits and security policy enforcement.
