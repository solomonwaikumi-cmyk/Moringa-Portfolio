## 📝 Description
An introductory room covering the daily responsibilities of a Security Analyst, including log analysis, monitoring, and incident response.

## 🛠️ Key Concepts Learned
* **Log Monitoring:** Identifying suspicious activity within a dashboard.
* **Incident Analysis:** Determining if an alert is a True Positive or False Positive.
* **Malware Triage:** Understanding the basics of how suspicious files are handled.

## 🚀 Methodology & Tasks
### Task 1: Monitoring and Logging
* Analyzed a simulated dashboard to identify a spike in login failures.
* **Command/Action used:** Filtered logs by "Failed Login" and "IP Address."

### Task 2: Malware Triage
* Learned the importance of sandboxing when analyzing suspicious email attachments.

## 💡 Reflection
This room bridged the gap between theoretical security and the practical tools used in a SOC environment. It reinforced the importance of attention to detail when monitoring network traffic.
