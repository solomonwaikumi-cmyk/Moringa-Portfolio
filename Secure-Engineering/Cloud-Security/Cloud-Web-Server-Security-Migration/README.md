# ☁️ Cloud Web Server Security Migration Report

Welcome to this Cloud Security Lab focused on securing and migrating a web server into a hardened cloud environment.

## 🔍 Overview
This project documents the process of assessing, hardening, and migrating a web server to improve cloud security posture and reduce exposure to common threats.

- Apply hardening controls
